# Phase 27.0 — No-React Lite Pro Core Shell

This phase resets the UI base to a freeze-safe No-React shell for Android Chrome + Termux.

## Included

- Core dashboard
- Sidebar + mobile bottom nav
- Module registry
- Safe module rendering
- Debug/error panels
- Health check
- Profile form
- Storage backup/restore
- PWA manifest + safe service worker
- Dependency-free backend health API
- Static frontend launcher

## Not Included Yet

- Full Kundli calculation
- Dasha calculations
- Premium PDF report
- APK wrapper

These will be added as plugins after Phase 27.0 passes interaction testing.
