# Books — Data Folder

> Phase 31 · Research Source Registry
> Status: **EMPTY BY POLICY.**

## What lives here

**Nothing is committed here.** This folder is a placeholder that documents the
policy for book/text references. Do **not** add any book content to this repo.

## Forbidden in this folder (and the whole repo)

- ❌ Copyrighted PDFs / book scans.
- ❌ Verbatim translation text of any granth (BPHS, Brihat Jataka, Saravali, etc.).
- ❌ Pirated or scraped book content.

## Where references live instead

- Classical texts are cataloged (titles + citation pointers only) in
  `docs/research/granth-list.md`.
- License status per text is tracked in `docs/research/source-inventory.md`
  and `docs/research/legal-risk-register.md`.

See `docs/research/research-workflow.md` for the full policy.
